# Data Analysis Portfolio

This repository is a collection of data analysis projects developed during my university studies and in my personal time. Each project explores different datasets, methodologies, and tools, with the goal of building a solid foundation in statistical analysis, machine learning, and data visualization.

The projects are written in **R** and **Python**, and cover topics such as time series forecasting, regression modeling, and exploratory data analysis. Each project includes a dedicated report or notebook with code, visualizations, and interpretation of results.

This portfolio reflects my ongoing learning process and my interest in turning raw data into meaningful insights.
