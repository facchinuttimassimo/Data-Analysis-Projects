# ARIMA Models on U.S. Industrial Production Index

This repository contains a Jupyter Notebook written as part of my studies.  The notebook analyzes the **U.S. Industrial Production Index** time series using **ARIMA models** (AutoRegressive Integrated Moving Average).
The dataset used is the **U.S. Industrial Production Index**, a macroeconomic indicator published by the Federal Reserve.

## Topics Covered
- Time series exploration and visualization
- Stationarity tests (e.g. ADF test)
- ARIMA model identification, estimation and diagnostics
- Forecasting the Industrial Production Index
